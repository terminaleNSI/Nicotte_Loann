----------------------------------------

Lancer Game_Laucher.py

----------------------------------------

Touches :
Z : aller en haut
Q : aller a gauche
S : aller en bas
D : aller a droite
E : ouvrir l'inventaire
T : ouvrir le chat

----------------------------------------

Commande dans le chat:

/spawn {slime;golem;wyvern;archer} {quantité} : pour faire apparaitre une entité, un monstre
/heal : pour remplir la bar de vie du joueur
/killall : pour tuer toutes les entitées de la salle
/resetchest : pour réinitialiser les coffre de la salle
/give {item_name} {quantité}
/collect {loot_name} : pour ramasser tous les objets par terre (all pour tous les items)
